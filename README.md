# Pokedex coding challenge: React app

## Installation:

- clone repo
- npm install

Run local:

- npm run dev

Unit Tests:

- npm run test

End to end tests:

- npm run test:e2e

Technologies used:

- Typescript
- React
- Cypress
- Jest
- Testing-library
- SCSS
- Component library: culotta-lib

Deployment:

- Github Actions
- Deployment to Netlify
